# show eth Address from local keyStore folder.
## mac环境编译:
```bash
chmod 700 ./build/keyManager.sh
./build/keyManager.sh
```

## 使用:
在命令行执行编译后的可执行文件
```bash
./bin/ethKeyManage
```
